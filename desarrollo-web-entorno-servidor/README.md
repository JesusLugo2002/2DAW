# Desarrollo web en entorno servidor - Introducción

Se refiere a la creación y gestión de la parte de una aplicación web que se ejecuta en el servidor, en lugar de en el navegador del usuario. Algunos puntos clave son:

__Lenguajes de Programación__: Los lenguajes más comunes incluyen PHP, Python, Ruby, Java y Node.js.

__Bases de Datos__: Las aplicaciones del lado servidor suelen interactuar con bases de datos para almacenar y recuperar información. Ejemplos de sistemas de gestión de bases de datos son MySQL, PostgreSQL y MongoDB.

__Seguridad__: El desarrollo del lado servidor incluye la implementación de medidas de seguridad para proteger los datos y la integridad de la aplicación.

__Generación Dinámica de Contenido__: Permite crear contenido dinámico que se adapta a las necesidades y acciones del usuario, como mostrar información personalizada.

__Comunicación Cliente-Servidor__: Utiliza protocolos como HTTP/HTTPS para la comunicación entre el cliente (navegador) y el servidor.
